<?php
var_dump('11'); var_dump 라고 함수는 입력값을 출력해주는데 그입력값에 데이터 값도 알려주 (11)입력하면 출력하면서 그것의 데이터탑입이 int 정수라는것을 알려주 
?> 

그리고 11을 '11'하면은 더이상 정수가 아니라 문자열이라서 string (2) "11" 로나오고  11을 출력동시에 11이 문자열이고 2개의 문자로 이루어저있다알려주는 것이 var_dump

(1+1)=2  더하는 연산자 

(1==1); true 연산자

(1==2) false 연산자 

false true 하나가 되고  한마디로 불리언 이되는 연산자 표현식이다.
