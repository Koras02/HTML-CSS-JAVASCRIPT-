
index.html 파일을 만들고 
<!doctype html>
<html>
    <body> 
       현재시간
      </body>
</html> 

하면 현재시간이 나오는데 

index.php 파일을 추가한뒤 
<!doctype html>
<html>
  <body>
     <?php
        echo date('Y-m-d H:i:s')
     ?>
    </body>
</html>
이부분을 추가하면 새로고침할떄마다 시간이 변경된다

이부분은 현재의 시간을출력해라 라는 php라는 언어의 문법임 

사용자가 요청한 파일이 index.html 이라면 그것을 웹서버는 파일을 읽어서 그대로 전송해줄뿐 즉 한번 웹서버 index html파일을 만들었다면 

언제나 똑같은 정보를 보여주게됨.

하지만 php가되면 웹서버는 자기가 처리할수없어서 php에게 위임 php는 index.php를 열어봄 

그결과 html 부분은 생략하고 <?php 라는 기호가나오면 php는 이제부터 php라는 언어의 문법이 등장하는것을 알고

그다음 등장하는 언어를 php언어로 해석해서 나타냄 

echo data ('Y-m-d H:i:s') 연도 연 월 일 시간 분 초의 형식의 따라 현재시간을 Echo(출력)해라라는 뜻 

php는 현재시간을 알아낸다음에 현재시간형식으로 바꿔버림 

바꾼부분을 웹서버에게 보내면 웹서버가 그것을 웹브라우저에게 보내주면 웹브라우저는 php부분이없는 순수한 html을 받을수있기떄문에

그냥 html 을 화면에 표시하게 됨.html은 한번만들면 언제나 동작하지만 php는 웹페이지를 생산할수있는 능력을 갖고있기떄문에 

리로드할떄마다 웹페이지를 프로그래밍쪽으로 동쪽으로 생성할수있음 

요약해서 

html은 한번만들면 언제나 똑같이 동작하지만 

php는 실시간으로 리로드할때마다 바뀜 

php 5.PHP의 데이터 타입부터 보면됨
