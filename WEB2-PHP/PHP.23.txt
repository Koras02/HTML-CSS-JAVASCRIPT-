함수의 형식 3/3 

함수의 출력값

sum 함수는 입력값을 통해다르게 적용 

sum 함수는 기능이2가지 하나는 입력값으로 들어온 두개의 값을 더한다 또하나는 화면에 출력한다.

무언가를 더해서 화면에 출력할때는 sum이라는 함수를 쓸때 편리 하지만 함수를 출력하는 기능까지 

포함시키면 어떤값을 더하고 더한결과를 화면에 출력하는것이아니 파일에저장하고 싶을떄&누군가에게

전송하고싶을때 &업로드하고싶을때

더하는 기능과 화면에 출력한다 퉁쳐버리면 더한다는 기능이 제한적 

함수만들때 함수는 하나의 기능만을 담는다라는 관점을 가져야함.. 

sum2 라는 함수는 함수가 자체적으로 결과를 출력해주는 프린트기능이 없지만 

실행하면 2 4 를 더한부분이 더한 결과 6이 되는 표현식을 만들기위해서는 

 print(sum2(2,4));  이렇게해서 6이라는 값을 화면에 출력시킬수있고 

email('이메일주소', sum2(2,4)); 
업로드하고싶다 upload('이메일주소', sum2(2,4));
파일에 저장하고싶다 file_put_contents('PHP.23.txt' , sum2(2,4)); txt형식으로 저장됨 

sum을 출력한다 가 내장되어있어 사용량 적음 

sum2는 내부적으로 동작한 결과를 리턴하기 떄문에 그값이 되기 떄매 사용됨 

<?php 
function sum($left, $right){
  return $left+$right;
}
print(sum2(2,4));
file_put_contents('result.txt', sum(2,4)); 메모장생성됨 
?>

return

6

함수시작할때 function 이 앞에오고 그뒤에 는 파라미터들이 온다 파라미터는 입력값이고 인자를 전달하면 
이 파라미터가 그함수안에서 사용되는 변수가 된다.최종적으로 그렇게 작업한결과를 return이라는 키워드 뒤에 
배치해주면 sum2함수를 실행한 결과는 리턴 뒤에있는 값이 된다. 유식하게는 표현식이라고 함 
function 함수자체는 statment 스테이트먼트다. 값이 되지 않지만 그것을 호출하면 개를 호출하면 리턴값이있다면
개는 값이되기 때문에 표현식이다.

함수의 이름이 오고 리턴이 만나는순간 그뒤에는 생략됨
